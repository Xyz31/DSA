### 5. Longest Palindromic Substring


```md

Given a string s, return the longest 
palindromic
 
substring
 in s.

 

Example 1:

Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer.
Example 2:

Input: s = "cbbd"
Output: "bb"
 

Constraints:

1 <= s.length <= 1000
s consist of only digits and English letters.

```

#### code

```cpp

class Solution {
public:
    bool check(int low,int high,string s){
        while(low<high){
            if(s[low] != s[high]) return false;
            low++;
            high--;
        }

        return true;
    }
    string longestPalindrome(string s) {
        int n=s.length();

        string ans = "";
        int len=0;

        for(int i=0;i<n;i++){
            for(int j=i;j<n;j++){
                int pal = check(i,j,s) ? j-i+1 : 0;

                if(pal > len){
                    cout<<pal<<" ";
                    ans = s.substr(i,pal);
                    len=pal;
                }
            }
        }

        return ans;
    }
};
```
